# TUAN Scriptum Icons · 築字師圖示集

![Preview](https://raw.githubusercontent.com/RossanniClover/tuan-scriptum-icons/main/preview.png)

Official icon assets for **築字師TUAN / TUAN: Scriptum** — a local-first writing and management tool for creators.  
These are functional UI icons designed for toolbars and feature buttons.

**築字師TUAN / TUAN: Scriptum** 的官方圖示資產。  
本集合為工具列與功能按鈕設計的功能型 UI 圖示，持續更新中。

🔗 [Product Page · 產品介紹](https://yuantuan.dev/tuan_scriptum.php)

## Usage · 使用說明

### ✅ Allowed · 允許

| Situation · 情境 | Example · 範例 |
|---|---|
| Use in your own software · 嵌入自己的軟體 | Embed icons in your app's UI · 將圖示用於自己開發的 App 介面 |
| Use in print or promotional materials · 用於印刷或宣傳素材 | Flyers, posters, editorial design · 傳單、海報、編輯設計 |
| Ship with your product · 隨產品一同發布 | Icons bundled inside your released app or game · 圖示包含在你上架的軟體或遊戲安裝包中 |
| Use in articles or videos · 用於文章或影片 | Screenshots or demos featuring these icons · 在介紹、評測或教學內容中展示這些圖示 |
| Recolor for your own use or redistribution · 改色後自用或散布 | Adapting colors to match your UI theme · 配合自己的介面配色調整顏色，但不得宣稱為築字師官方圖示 |
| Modify shapes and release as your own work · 改變形狀後作為自己的作品發布 | Creating a derivative icon set · 基於本圖示創作衍生圖示集，但不得宣稱來自築字師 |

### ❌ Not Allowed · 不允許

| Situation · 情境 | Example · 範例 |
|---|---|
| Re-upload original icons as a collection · 重新上傳原始圖示集 | Uploading this icon pack to Google Drive, Dropbox, or any platform · 將本圖示包上傳至 Google Drive、Dropbox 或其他平台供他人下載 |
| Mirror on other platforms · 搬到其他平台散布 | Publishing on npm, Figma Community, another GitHub repo · 發布至 npm、Figma Community、其他 GitHub repo |
| Sell as an icon pack · 打包販售 | Bundling and selling these icons as a standalone asset product · 將本圖示集作為獨立素材商品進行販售 |
| Claim modified versions as official TUAN assets · 宣稱改作版本為築字師官方圖示 | Redistributing recolored icons under the TUAN name · 以築字師名義散布改色或改形狀的版本 |
