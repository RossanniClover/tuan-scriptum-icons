# Icon Usage Guide · 圖示使用說明

## English

These icons are free to use. Here's what you can and cannot do:

### ✅ Allowed

| Situation | Example |
|---|---|
| Use in your own software | Embed icons in your app's UI |
| Ship with your product | Icons bundled inside your released app or game |
| Use in articles or videos | Screenshots or demos featuring these icons |

### ❌ Not Allowed

| Situation | Example |
|---|---|
| Re-upload as a collection | Uploading this icon pack to Google Drive, Dropbox, or any file-sharing platform for others to download |
| Mirror on other platforms | Publishing these icons on npm, Figma Community, another GitHub repo, or similar |
| Sell as an icon pack | Bundling and selling these icons as a standalone asset product |
| Modify and redistribute | Recoloring or reshaping icons and releasing them as a new set |

### 💬 Attribution

## 繁體中文

本圖示集免費使用，以下說明哪些行為被允許、哪些不被允許。

### ✅ 允許的使用方式

| 情境 | 範例 |
|---|---|
| 嵌入自己的軟體 | 將圖示用於自己開發的 App 介面 |
| 隨產品一同發布 | 圖示包含在你上架的軟體或遊戲安裝包中 |
| 用於文章或影片 | 在介紹、評測或教學內容中展示這些圖示 |

### ❌ 不允許的使用方式

| 情境 | 範例 |
|---|---|
| 重新上傳整包圖示 | 將本圖示包上傳至 Google Drive、Dropbox 或其他雲端平台供他人下載 |
| 搬到其他平台散布 | 將圖示發布至 npm、Figma Community、其他 GitHub repo 或類似平台 |
| 打包販售 | 將本圖示集作為獨立素材商品進行販售 |
| 改作後散布 | 修改顏色或造型後，作為新的圖示集對外發布 |
